# dotfile

## Usage

### dotfiles
- ```bash dotfilesLink.sh```

### Vim

- dein
  - ```mkdir -p ~/.vim/dein/repos/github.com/Shougo/dein.vim```
  - ```git clone https://github.com/Shougo/dein.vim.git ~/.vim/dein/repos/github.com/Shougo/dein.vim```
  - ```:call dein#install()```

### Ruby Gems
- local install
  - ```bash localRubyGems.sh```
